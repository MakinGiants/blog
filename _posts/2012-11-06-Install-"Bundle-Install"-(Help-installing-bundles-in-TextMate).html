Hi

If you want to install "Bundle Install" Bundle to TextMate do:

Run:

$mkdir -p ~/Library/Application\ Support/TextMate/Bundles
$svn co http://svn.textmate.org/trunk/Review/Bundles/GetBundles.tmbundle/

Then, in TextMate, go to
TextMate/Bundles/GetBundles/GetBundles
