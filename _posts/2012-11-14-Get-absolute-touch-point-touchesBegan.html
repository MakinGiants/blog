--- title: Get absolute touch point touchesBegan layout: post date: 2012-11-14 23:42:00 UTC updated: 2015-02-08 19:42:31 UTC comments: false categories: IOS --- Add the next method to code:

-(void) touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event {
    UITouch *touch = [[event allTouches] anyObject];
    CGPoint location = [touch locationInView:touch.view];
    NSLog(@"X: %f",location.x);//Print values for debug
    NSLog(@"Y: %f",location.y);
}
Now location.x and location.y you have the absolute touch point
